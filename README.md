# Basic-Nodejs-CRUD
This is a basic nodejs backend server with Mysql
- Able to add data to the database using json format.
- Get data from server as json format.
- Use JSON format

## Features
- Add data to database.
- Get data from database.
- Update specific data.
- Delete specific data.

## Technology

- Nodejs
- Mysql

## Installation
Install dependencies:
```sh
npm install
```
Start server:
```sh
node index.js
```

## Dependencies
- express
- mysql
- cors
- body-parser

## Port

```sh
localhost:3000
```
